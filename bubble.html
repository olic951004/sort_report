氣泡排序（Bubble Sort）

視覺化步驟演示

原理： 氣泡排序會重複比較相鄰的兩個元素， 若順序錯誤就交換，讓較大的數字像氣泡一樣慢慢浮到右邊。

時間複雜度：平均 / 最差 O(n²)，最佳（已排序）為 O(n)
第 0 輪
比較：0
交換：0
狀態：就緒
按「開始」觀看氣泡排序過程
function bubbleSort(arr) {
  for (let i = 0; i < arr.length-1; i++) {
    for (let j = 0; j < arr.length-i-1; j++) {
      if (arr[j] > arr[j+1]) {
        [arr[j],arr[j+1]] = [arr[j+1],arr[j]];
      }
    }
  }
}
