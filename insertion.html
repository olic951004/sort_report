插入排序（Insertion Sort）

視覺化步驟演示

原理： 插入排序會將元素逐一插入到已排序的區域中。 每一輪排序，已排序區間向右擴大一格。

⏱ 時間複雜度：平均 / 最差 O(n²)，最佳（幾乎已排序）為 O(n)
第 0 輪
比較：0
移動：0
狀態：就緒
數量
速度 700ms
function insertionSort(arr) {
  for (let i = 1; i < arr.length; i++) {
    let key = arr[i];
    let j = i - 1;
    while (j >= 0 && arr[j] > key) {
      arr[j + 1] = arr[j];
      j--;
    }
    arr[j + 1] = key;
  }
}
